How to use it?

This simulator is really easy to use. This is a step-by-step guide that will show you how to build your first, simple network. If you need some general informations about IP protocol, you will find it here:

Buttons

This button can be used to start simulation after you finish configuration.

This button stops simulation and enables back building network.

This button creates a new simulation. It does not save your current workspace state so be careful, because you will not be able to bring back unsaved changes.

You can save your simulation using this button. Your saved simulation will appear in your account profile.

Those two buttons can be used to add router and client node to the workspace.

Link button turns on a drawing mode. It means that you can create links between nodes. To do it, click once on any two nodes in the workspace.

This button enables pinging nodes. After clicking Ping button, click source node and then click destination node.

This button deletes last chosen node.
